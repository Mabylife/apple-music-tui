# Apple Music TUI - 使用指南

## 基本操作

### 導航

- `↑` / `↓` - 上下選擇項目
- `→` - 進入選中項目的下一層（或播放）
- `←` / `Backspace` / `Delete` - 返回上一層

### 搜尋

- `Tab` - 進入搜尋模式
- 輸入關鍵字
- `Enter` - 執行搜尋
- `ESC` - 取消搜尋

### 命令

- `:` - 進入命令模式
- `:q` / `:quit` / `:qa` - 退出應用
- `:home` - 返回推薦內容（第一層）
- `ESC` - 取消命令

## 項目類型與行為

| 圖示 | 類型                | 按右鍵行為       | note                              |
| ---- | ------------------- | ---------------- | --------------------------------- |
| 󰝚    | Track (歌曲)        | **直接播放**     |
| 󰀥    | Album (專輯)        | 展開顯示曲目列表 |
| 󰲸    | Playlist (播放清單) | 展開顯示曲目列表 |
| 󰐹    | Station (電台)      | **直接播放**     | And will be entering station mode |
| 󱍞    | Artist (藝人)       | 顯示分類選單     |

## Artist (藝人) 導航

選擇藝人後會顯示三個分類：

```
Artist (藝人)
  ├─ 󰝚 Top Tracks (熱門歌曲)
  │   └─ 按右鍵 → 顯示 20 首熱門歌曲
  │
  ├─ 󰀥 Albums (專輯)
      └─ 按右鍵 → 顯示所有專輯（由新到舊）

```

## 範例使用流程

### 1. 瀏覽推薦內容

```
啟動應用 → 自動載入推薦內容
使用 ↑↓ 瀏覽
按 → 選擇項目
```

### 2. 搜尋音樂

```
按 Tab → 輸入 "coldplay" → 按 Enter
使用 ↑↓ 選擇結果
按 → 進入項目
```

### 3. 探索藝人

```
搜尋或選擇藝人
按 → 進入
選擇 "Top Tracks" / "Albums"
按 → 查看列表
選擇歌曲 → 按 → 播放
```

### 4. 快速返回

```
在任何層級
按 : → 輸入 home → 按 Enter
立即返回推薦內容
```

## 提示

- 📍 **Loading..** 表示正在載入資料
- 🎵 選擇歌曲後會立即開始播放
- 📻 電台會串流播放，無固定曲目列表
- 🔙 可以隨時用 `←` 返回上一層
- 🏠 使用 `:home` 快速回到首頁

## 系統需求

- Node.js
- Cider 2 運行在 `localhost:10767`
- 終端支援 Nerd Fonts（顯示圖示）
